Functional Area : Using Java Classes

Java classes can be represented by a literal which is a concatenation of identifiers and . 's, in appropriate context. They can also be obtained and assigned to constants or variables with the javaclass operator. Static methods and data members of Java class literals are accessed or invoked with the :: operator. Those of class constants or variables can be used like any regular objects. The Java class itself for a Java class object is obtained via its .class data member.
