JudoScript Reference

Structurally, JudoScript language has three major components:

  1. Syntax and Semantics
  2. System Functions
  3. Built-In Object Types
Functionally, JudoScript language contains various functional areas:
  • Functional Areas
Each function area may have syntactic, system functions and object support.

This comprehensive documentation is the complete, most current reference to the entire language. On this site, the documentation is updated with each minor release of the language. For your release, refer to the documentation that came with that release. The current release information is available on the download page.
