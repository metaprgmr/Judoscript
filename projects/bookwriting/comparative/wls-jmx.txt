Weblogic server = managed resources + managed beans

Weblogic server beans = configuration MBeans + runtime MBeans


<SECTION title="Configuration MBeans">

The admin server hosts all the configuration MBeans for all managed
resources on all server instances.

The admin server also saves changes to these beans.

Configuration MBeans in the admin server are copied to each managed
server. The replicas are called local configuration MBeans.
