Install and Run

Now that you are reading this page, you have successfully obtained the software and have it unpacked. Check the following list before run any Jamaica programs:

  1. Your JVM version is JDK 1.3 or equivalent or highter.
  2. Put file jamaica.jar in your classpath.
  3. Download the BCEL-5.1 (or up) software from http://jakarta.apache.org or the ASM-1.4.2 software from http://asm.objectweb.org, unzip and put their executable jar files in the classpath.
  4. Run com.judoscript.jamaica.Main with a source file name.

Release Notes

Jamaica version 1.0.

Changes

[January 2005 Release] This release fixes a typo related to BCEL.

[April 2004 Release] This release adds a built-in support for ASM and some minor changes to the JavaClassCreator API (to make it simpler).

[March 2004 Release] This is the initial release of the Jamaica language.

Documentation and Examples

The documentation for the Jamaica language is in the article, Jamaica, the Java Virtual Machine Macro Assembler.

The examples are includes the following:

  1. Testing for all Jamaica language features, and
  2. The use of the JavaClassCreator Java API that dynamically creates Java classes within Java software.
