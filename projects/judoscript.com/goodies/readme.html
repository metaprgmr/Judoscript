Check out JudoScript, the scripting tool for the Java platform! It is free and open source under LGPL.

JudoScript is also a great scripting tool for Windows. It supports scripting for COM/ActiveX and registry, now, and will support even more.

JudoScript is a powerful programming language and also a highly function language, with intimate support for, e.g.,

  • RDBMS scripting via JDBC
  • XML (SAX, DOM and XSLT) scripting
  • SGML scripting
  • Archiving, compression and encryption
  • O.S. shell scripting
  • Internet scripting
  • Java and Java GUI scripting
  • Windows scripting
  • etc.
