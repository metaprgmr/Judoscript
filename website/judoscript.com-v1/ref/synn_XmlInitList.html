Semantics : XmlInitList

XmlInitList  ::=  with ( IDENTIFIER ( = true | false )? | xmlns ( : IDENTIFIER )? = STRING_LIT ),+

Used by:  

The XML initialization parameters include these boolean options: "namespace", "validate", "schema", "ignoreWhite" and "ignoreComment". You can also declare a XML namespace ID with the xmlns option, so that tags starting with the ID and ":" will use this namespace.
