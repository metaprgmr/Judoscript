Useful Web Links

  • JDBC Scripting 1 -- An article on JavaWorld.com.
  • JDBC Scripting 2 -- An article on JavaWorld.com.
  • Learn to speak Jamaican -- An article on JavaWorld.com.
  • JudoScript Project on SourceForge -- We are moving development resources and distributions more and more to that location.
  • Sun's Java.NET Site -- A Java resource portal.
  • JavaCC -- The Java Compiler Compiler tool used by JudoScript and most other Java scripting languages for parser generation.
  • BSF -- Bean Scripting Framework -- A standard framework to incorporate scripting languages into Java software. JudoScript is listed in its projects page.
  • JCom -- Java-COM Bridge -- Used by JudoScript for its COM scripting support.
  • JACOB and its User Group -- This is another popular Java-COM bridge API. Its User Group is active and you can find a lot good examples to deal with Microsoft products, etc.
  • Tim Endres' Java TAR Package -- Used by JudoScript to support its TAR functionality.
  • Enterprise Distributed Technologies' Java FTP Client Library: Used by JudoScript to support its FTP functionality.
  • ISNetworks SSH/SCP package -- Used by JudoScript to support its SSH/SCP functionality.
  • ACME Labs -- Has many useful Java packages.
  • Languages for the Java VM -- A popular site listing all Java-based languages.
  • Sun's Java site -- How can we forget this site? It has all the latest Java standard and extension APIs.
  • A JavaWorld Article -- This three-part article compares JACL, Jython, Rhino and BeanShell. The author must have had a hard time to discover sensible differences between them.
