Functional Area : Object and Class

In JudoScript, every value is an object. Users can also define their own classes, which by default extends Object . A class can also extend OrderedMap , TreeNode or anoother user-defined class.

Objects are created with the new operator. including general built-in types such as container data structures and Object , Orderedmap , TreeNode , TableData and Date . Any Object -derived classes can take named initializers.
